# Pokedex Clone
Este proyecto trata de imitar el comportamiento de una Pokedex. La idea es mostrar información sobre los distintos Pokémon, sus habilidades, tipos y los movimientos que aprenden. También se tiene en consideración su cadena evolutiva, y que hay Pokémon que tienen múltiples evoluciones.

## Tecnologías
Se implementarán más tecnologías con el pasar del tiempo.
- SQL Server
